# 🛒 E-comerce_MVC

Um projeto de **e-commerce desenvolvido em PHP utilizando arquitetura MVC**, criado com o objetivo de aprofundar conhecimentos em desenvolvimento web, organização de código e boas práticas de programação orientada a objetos.

> ⚠️ **Status do projeto:** Em desenvolvimento. Este repositório faz parte do meu portfólio e continuará recebendo melhorias e novas funcionalidades.

## ✨ Funcionalidades

* Login e autenticação
* Perfil do usuário
* Catálogo de produtos
* Página individual de produtos
* Carrinho de compras
* Finalização de pedidos
* Integração com Mercado Pago
* Painel administrativo
* Edição de produtos
* Upload de imagens
* Registro de logs da aplicação

---

## 🛠️ Tecnologias utilizadas

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Composer
* Mercado Pago SDK
* Apache

## 📂 Estrutura do projeto

Essa organização facilita a manutenção do código, separando responsabilidades entre:

* **Models** → acesso aos dados
* **Views** → interface do usuário
* **Controllers** → regras de negócio

```
E-comerce_MVC/
│
├── Motor/
├── vendor/
├── config.php
├── composer.json
├── index.php

```
